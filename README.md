# Sofrim
Omer Notification App



## Installation

- Clone or download.
- Run ```npm i```
- Install platforms:
    - Android: ```ionic cordova platform add android```
    - iOS: ```ionic cordova platform add ios```
- Run:
    - Web: ```ionic serve```
    - Android: ```ionic cordova run android```
    - iOS: ```ionic cordova run ios```
